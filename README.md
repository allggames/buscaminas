# Buscaminas

Este es un proyecto de ejemplo para construir el juego de Buscaminas en una cuadrícula 5x5 con minas aleatorias.

### Características principales
- Tamaño de la cuadrícula: 5x5
- Número de minas: 5
- Generación y funcionalidad paso a paso para facilitar el desarrollo y aprendizaje.

### Instrucciones
El objetivo del juego es revelar todas las casillas sin pisar una mina.
